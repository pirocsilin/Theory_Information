    Изучение дисциплины «Теория информации» ставит следующие цели: получение знаний о количественных мерах информации сообщений и методах кодирования информации; приобретение умений разработки программ для решения задач оптимального кодирования информации. Изучение данной дисциплины базируется на материале курсов «Дискретная математика», «Структуры и алгоритмы обработки данных», «Теория вероятностей и математическая статистика», «Программирование». Дисциплина является предшествующей для большинства дисциплин, в том числе: «Защита информации», «Сети ЭВМ и телекоммуникации», «Представление графической информации».

    В процессе изучения данной дисциплины Вам необходимо:

    1. Изучить конспект лекций;
    2. Выполнить лабораторные работы;
    3. Выполнить контрольную работу;
    4. Сдать экзамен/зачет.

    Внимательно прочтите следующие рекомендации:

    1. При изучении курса «Теория информации» кроме электронного конспекта лекций рекомендуется использовать учебники из списка литературы, а также любую литературу по программированию на языках высокого уровня, учебники для ВУЗов по теории информации и теории кодирования.
    2. Освоение курса «Теория информации» невозможно без практики на компьютере. Для успешного изучения данного курса необходимо иметь навыки программирования на языках высокого уровня. Лабораторные работы выполняются на языке программирования С/С++, можно использовать любой доступный компилятор.
    3. Следует сначала выполнить и сдать преподавателю все лабораторные работы, и только после этого приступать к выполнению контрольной работы.
    4. В конспекте лекций приводятся алгоритмы на псевдокоде (расшифровку псевдокода см. в приложении), а также примеры применения методов кодирования, которые могут быть полезны при выполнении контрольной и лабораторных работ.
    5. Выполненные задания должны быть высланы со страниц нашего сайта. С порядком отправки выполненных работ можно ознакомиться в инструкции по отправке, доступной по ссылке.
    6. После проверки высланных работ преподавателем Вы получите рецензию с оценкой и возможными замечаниями, которые следует устранить. Если работа не зачтена, то после доработки и устранения ошибок Вы должны повторно выслать работу на проверку.
    7. Контрольная работа выполняется только после выполнения всех лабораторных работ в указанном в конспекте порядке.
    8. Оценка за курс выставляется по результатам выполнения лабораторных работ, контрольной работы и теста(экзамен/зачет).

    Получить необходимую консультацию можно по адресу: e_match@ngs.ru.
